## Beni Digital Store is a demo store that you can buy and add/remove from the cart, design with bootstrap.

**made with: Angular**

[demo](https://beni-digital-store.vercel.app/)

## Usage

#

### Install Angular CLI

```bash
npm install -g @angular/cli
```

### Install dependencies

```bash
npm install
```

### Run

```bash
ng serve --open
```
